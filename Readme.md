<h4>Day 1 - Arrays</h4>

<h5>Concepts</h5>

<p>Arrays are used to store multiple values of same data type in a single variable.

<h5>Code</h5>

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    for(int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    return 0;
}
```

<h5>Output</h5>

```
1 2 3 4 5
```

<h5>Time Complexity</h5>

```
O(n)
```
